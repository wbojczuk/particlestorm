particleStorm.js
Why Use particleStorm.js?

Let's be honest. There are hundreds, or thousands of javascript particle frameworks, many with way more features than this one. This script was made for performance. All code is simple, easy to read and follows ES6 standards. Usage is even more easy, this can be seen in the documentation below.

How Is The Performance Better?

  • • DOM manipulation is kept to a minimum and the only thing EVER injected to it is one canvas element.
  • • The code is simple, clean, and follows ES6 practices
  • • NO dependencies.

Methods/Settings
Methods:
start(); Starts the script if off, and if paused, can unpause.

stop(); Removes any exisiting particles, removes the canvas from the DOM and kills any event listeners used.

pause(); Freezes any extisting particles and pauses any script functions.

resume(); Unfreezes script if paused or starts script if off.


Settings:
minXSpeed: Minimum speed for particles on the X-Axis.

maxXSpeed: Maximum speed for particles on the X-Axis.

minYSpeed: Minimum speed for particles on the Y-Axis.

maxYSpeed: Maximum speed for particles on the Y-Axis.

minSize: Minimum size for particles.

maxSize: Maximum size for particles.

zIndex: z-index css attribute for the canvas element.

particleColor: Color setting for particles passed as a string value.

strokeColor: Stroke color setting, default value is null which disabled the stroke.

strokeSize: Size for stroke, if enabled.

amount: Amount of particles.

autoStart: Setting for whether to automatically start the script or not.
